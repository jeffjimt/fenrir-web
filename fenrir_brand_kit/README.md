# Fenrir Soluciones Digitales - Brand Kit inicial

Este paquete contiene recursos PNG extraídos y preparados a partir del board visual aprobado en la conversación.

## Archivos principales

- `png/fenrir-brand-board.png`: board visual original convertido a PNG.
- `png/fenrir-logo-full-transparent.png`: logo completo color con fondo transparente.
- `png/fenrir-logo-full-white-blue-transparent.png`: versión para fondos oscuros.
- `png/fenrir-logo-full-with-tagline-transparent.png`: logo completo con frase de apoyo.
- `png/fenrir-isotipo-color-transparent.png`: isotipo F color con fondo transparente.
- `png/fenrir-isotipo-white-blue-transparent.png`: isotipo para fondo oscuro.
- `png/fenrir-isotipo-mono-navy-transparent.png`: isotipo monocromático navy.
- `png/fenrir-isotipo-mono-white-transparent.png`: isotipo monocromático blanco.
- `png/fenrir-favicon-32.png`, `64`, `180`, `192`, `512`: favicons/app icons.
- `png/fenrir-og-image-1200x630.png`: imagen para compartir en WhatsApp/redes.
- `prompts/codex-main-brief.md`: prompt principal para Codex.
- `prompts/codex-followups.md`: prompts de seguimiento.
- `prompts/contexto-negocio-y-marca.md`: resumen de la estrategia y marca.

## Nota importante

Estos PNGs salen de un mockup raster. Sirven para arrancar la web, prototipos y primeras pruebas comerciales. Para una marca definitiva conviene recrear el isotipo en SVG/vector limpio en Figma/Illustrator/Inkscape y exportar desde ahí.


## Archivos limpios recomendados

Para la web usa preferiblemente estos archivos generados de forma vectorial/limpia:

- `png/fenrir-logo-horizontal-clean.png` para header en fondo claro.
- `png/fenrir-logo-horizontal-white-blue-clean.png` para header/footer en fondo oscuro.
- `png/fenrir-logo-full-clean.png` para secciones grandes o presentaciones.
- `png/fenrir-logo-full-white-blue-clean.png` para fondos navy.
- `png/fenrir-isotipo-color-clean.png` para iconos grandes.
- `png/fenrir-isotipo-white-blue-clean.png` para fondos oscuros.
- `png/fenrir-isotipo-on-navy-square.png` para perfiles o avatar.
- `png/fenrir-favicon-clean-32.png`, `64`, `180`, `192`, `512` para favicon/app icons.
- `svg/fenrir-isotipo-clean.svg` si necesitas vector simple.

Los archivos sin “clean” son extracciones del mockup y sirven como referencia visual, pero pueden contener artefactos porque provienen de una imagen JPG.
