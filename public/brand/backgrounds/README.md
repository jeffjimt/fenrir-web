# Fenrir Hero Background Resources

Estos recursos reemplazan el video HLS por fondos estáticos o CSS-only más limpios, suaves y rápidos.

## Recomendación
Usa primero `fenrir-hero-blue-ribbons.png` como fondo principal del hero.

Ruta sugerida dentro del proyecto:

```text
public/brand/backgrounds/fenrir-hero-blue-ribbons.png
public/brand/backgrounds/fenrir-hero-blue-ribbons.svg
public/brand/backgrounds/hero-background.css
```

## Opciones

1. `fenrir-hero-blue-ribbons.png` / `.svg`
   - Más cercano a la referencia azul con cortinas/paneles verticales.
   - Recomendado para el hero principal.

2. `fenrir-hero-aurora-grid.png` / `.svg`
   - Más tecnológico y sobrio, con grid sutil.
   - Bueno si quieres algo más elegante y menos intenso.

3. `fenrir-hero-neural-soft.png` / `.svg`
   - Más abstracto/tech, parecido a redes/neural, pero menos ruidoso que video.
   - Útil si quieres mantener una sensación IA sin movimiento pesado.

4. `hero-background.css`
   - Alternativa sin imágenes ni video.
   - Usa gradientes CSS, líneas verticales y glows.

5. `codex-replace-video-prompt.md`
   - Prompt listo para Codex para reemplazar el video por estos fondos.
